# 启动命令

## 验证标准

- FastAPI 后端可在本机启动，并能访问 `http://127.0.0.1:8000/docs`。
- Vite 前端可在本机启动，并能访问 Tauri 配置里的 `http://127.0.0.1:5173`。
- Tauri 桌面壳可拉起，并复用 `src-tauri/tauri.conf.json` 中的 `beforeDevCommand` 自动启动前端。

## 首次环境准备

在项目根目录执行：

```bash
cd /Users/liudiudiu/My_Project/麦搜
npm install
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -e ".[dev]" uvicorn
```

如果本机还没有 Tauri CLI，补一次开发依赖：

```bash
cd /Users/liudiudiu/My_Project/麦搜
npm install -D @tauri-apps/cli
```

## 启动全套服务

### 终端 1：启动 FastAPI 后端

```bash
cd /Users/liudiudiu/My_Project/麦搜
source .venv/bin/activate
uvicorn app.main:app --app-dir backend --host 127.0.0.1 --port 8000 --reload
```

### 终端 2：启动 Tauri 桌面端

```bash
cd /Users/liudiudiu/My_Project/麦搜
npm exec tauri dev
```

`npm exec tauri dev` 会读取 `src-tauri/tauri.conf.json`，并通过 `beforeDevCommand` 自动执行 `npm run dev -- --host 127.0.0.1` 拉起 Vite。

## 只启动浏览器前端

不需要桌面壳时，用这一条：

```bash
cd /Users/liudiudiu/My_Project/麦搜
npm run dev -- --host 127.0.0.1
```

## 健康检查

后端接口文档：

```bash
open http://127.0.0.1:8000/docs
```

前端页面：

```bash
open http://127.0.0.1:5173
```

## 停止服务

在对应终端按：

```bash
Ctrl+C
```

如果端口被占用，先查看占用进程：

```bash
lsof -nP -iTCP:8000 -sTCP:LISTEN
lsof -nP -iTCP:5173 -sTCP:LISTEN
```
