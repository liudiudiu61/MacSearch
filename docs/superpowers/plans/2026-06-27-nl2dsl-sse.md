# NL2DSL SSE Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the smallest testable FastAPI NL2DSL endpoint with configuration-loaded prompt metadata, strict Pydantic DSL validation, and SSE progress events.

**Architecture:** Prompt and DSL options live in `prompts/nl2dsl/v1.json`; Python code loads that file rather than embedding business rules. The API layer exposes a streaming endpoint and delegates query conversion to a focused service.

**Tech Stack:** FastAPI, Pydantic v2, pytest, JSON configuration.

from fastapi.testclient import TestClient

from app.main import create_app
from app.services.nl2dsl import load_prompt_config, parse_nl_query


def test_loads_prompt_config_from_json_file():
    config = load_prompt_config()
    assert config.version == "v1"
    assert "received" in config.sse_events


def test_parse_nl_query_returns_validated_dsl_from_configured_terms():
    dsl = parse_nl_query("找 Markdown 里的架构设计")
    assert dsl.query_text == "找 Markdown 里的架构设计"
    assert any(filter_item.field == "content" for filter_item in dsl.filters)


def test_nl2dsl_stream_emits_configured_sse_events():
    client = TestClient(create_app())
    with client.stream("POST", "/api/nl2dsl/stream", json={"query_text": "找 Markdown 里的架构设计"}) as response:
        body = response.read().decode("utf-8")
    assert response.status_code == 200
    assert response.headers["content-type"].startswith("text/event-stream")
    assert "event: received" in body
    assert "event: analyzing" in body
    assert "event: generated" in body
